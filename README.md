### Spring boot sample
- Build
```
gradle build
```
- Run
```
grandle bootRun
java -jar build/libs/spring-boot-sample-0.0.1-SNAPSHOT.jar
```
- Test summary
```
file://<path to spring boot sample>build/reports/tests/index.html
```
